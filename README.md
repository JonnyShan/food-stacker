# Food Stacker — Uber Eats Side Quest

A tetromino-style "pack the delivery bag" mini-game. Slide, rotate and drop
food items to pack the Uber Eats bag with no gaps.

**Play:** https://jonnyshan.github.io/food-stacker/

Standalone build with the redesigned 7-piece icon set (sushi, coffee, toilet,
Uber Eats bag, flowers, takeout, fruit) — uniform tiled art, single HTML file.
